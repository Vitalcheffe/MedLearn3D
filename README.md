## What

A medical learning app with 3D anatomy visualization, AI-generated quizzes (via Gemini), and progress tracking. Built with React, Three.js, Firebase, and i18n (French + English).

## Features

- **3D anatomy viewer** — multiple layers (skin, muscle, skeleton, cardiovascular, nervous, etc.)
- **AI quizzes** — generated by Gemini, multiple difficulty levels
- **Progress tracking** — XP system, levels, badges
- **Auth** — Google sign-in via Firebase
- **i18n** — French and English
- **Dashboard** — recent activity, quiz history, learning progress charts

## Stack

- React + TypeScript
- Three.js + React Three Fiber
- Firebase (Auth + Firestore)
- Google Gemini API (quiz generation)
- Recharts (progress charts)
- Lucide React (icons)
- i18next (translations)

## Run it

```bash
npm install
cp .env.example .env.local   # add your Firebase + Gemini keys
npm run dev
```
